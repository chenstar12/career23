```python
m, n = map(int, input().split())
# 输入一维数组
li = list(map(int, input().split()))

# 第一行输入行数n，然后输入每行的数据
n = int(input())

```

## n行多组输入

```python
n = int(input()) # 确定行数n
res = []
for _ in range(n):
    try:  # 避免EOF问题
        line = input() # 默认str类型
    except:
        break
    a,b = line.split() # 一行有多个输入
    res.append(int(a)+int(b)) # 类型转换
```

## 不确定行数
```python
res = []
while True: # 此处行数未确定，需要while True循环
    line = input() # 输入就是input
    if line != "":
        a,b = line.split()
        res.append(int(a)+int(b))
    else:
        break
```

## 从文件读取

```python
fin = open("in.txt", "r");
fout = open("out.txt", "w");

line = fin.readline()
while fin != EOF:
    fout.write(line)
    line = fin.readline()

fin.close()
fout.close()
```
